# Fundamentos de Ciberseguridad

Repositorio creado como parte de mi proceso de aprendizaje en ciberseguridad.

## Objetivo
Adquirir conocimientos básicos sobre seguridad informática, amenazas comunes y buenas prácticas para la protección de la información.

## Contenido
- Conceptos básicos de ciberseguridad
- Tipos de amenazas (phishing, malware, ransomware)
- Buenas prácticas de seguridad
- Apuntes del curso de Ciberseguridad (Coursera)
